# Smart Expense Management Application

## 📌 Description

Smart Expense est une application mobile développée avec Flutter et Spring Boot permettant aux étudiants de gérer efficacement leurs dépenses personnelles et leurs revenus mensuels.

L'application offre :

- Gestion des transactions
- Suivi du budget mensuel
- Statistiques financières
- Graphiques interactifs
- Alertes de dépassement de budget
- Interface moderne et intuitive

---

# 🛠 Technologies utilisées

## Frontend
- Flutter
- Dart
- fl_chart

## Backend
- Spring Boot
- Spring Data JPA
- H2 Database
